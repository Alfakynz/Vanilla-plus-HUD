# 🌟 Vanilla+ HUD

Vanilla+ HUD is a better alternative to the Vanilla HUD, keeping the Vanilla style.

## ✨ Compatibility

- [AppleSkin](https://modrinth.com/project/EsAfCjCV)
- [Feathers](https://modrinth.com/project/6Na0gn2c)

## 📦 Installation

You can download Vanilla+ HUD on [Modrinth](https://modrinth.com/project/BSuopNBj) or [CurseForge](https://www.curseforge.com/minecraft/texture-packs/pure-hud)

## 📦 Modpack

Feel free to include Vanilla+ HUD in modpacks.
